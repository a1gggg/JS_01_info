Базові математичні оператори

Термінологія:
 Операнд - те до чого застосовується оператор(аргумент). // наприклад у виразі 1+1, операндами будуть 1 та 1
 Оператор(аргумент) - дія, у вигляді символу (+, -, *, / і т.д.), яка застосовується до операнда. // наприклад у виразі 1+1, операндом буде унарний +
 Оператори є двох видів:
Унарні - застосовується до одного операнда. // У виразі let x = 1; x = -x; alert( x ); // -1, було застосоване унарне заперечення/
 Бінарний - оператор застосовується до двох операндів. // У виразі 1 + 1, бінарний оператор "+" застосовується до двох операндів 1 та 1.


Математичні оператори в JavaScript:
 Бінарний + (додавання),
 Унарний + (перетворення на number),
 Віднімання -,
 Множення *,
 Ділення /,
 Остача від ділення %,
 Піднесення до степеня **.


Бінарний + (додавання)
 Бінарний плюс об'єднує рядки:
  ("1" + "2") //об'єднає рядки та поверне "12"
  ( '1' + 2 ); // повертає "12" - перший string перетворює подальші значення (в даному випадку number) в string
  ( 2 + '1' ); // повертає "21" перший number об'єднується з наступним string та повертає string
  (2 + 2 + '1' ); // повертає "41" - оператори виконуються один за одним. Перший + додає два number, тому він поверне 4; а наступний оператор + вже додасть (об’єднає) попередній результат із string. У підсумку ми отримаємо рядок.
  ('1' + 2 + 2); // повертає "122" - перший операнд – string, тому компілятор також опрацьовує інші два операнди як string. Операнд 2 приєднується (конкатенується) до '1', тому в результаті буде '1' + 2 = "12", а потім — "12" + 2 = "122".
 Лише бінарний + працює з рядками так. Інші арифметичні оператори працюють тільки з числами й завжди перетворюють свої операнди на числа.
Приклади----------------
let apples = "2";
let oranges = "3";

alert( apples + oranges ); // "23", бінарний плюс об’єднує значення змінних, які є рядками
------------------------


Унарний + (перетворення на number)
 Альтернатива функції Number(...)
  let x = 1; alert( +x ); // повертає 1 - унарний + не впливає на числа.
  let y = -2; alert( +y ); // повертає -2 - унарний + не впливає на числа.
  ( +true ); // повертає 1 - Перетворює нечислові значення
  ( +"" ); // повертає 0 - Перетворює нечислові значення
Приклади----------------
let apples = "2";
let oranges = "3";

// обидва значення перетворюються на числа перед застосуванням бінарного плюса
alert( +apples + +oranges ); // 5

// довший варіант
// alert( Number(apples) + Number(oranges) ); // 5
------------------------


Віднімання -
( 6 - '2' ); // 4, '2' перетворюється на число (не так, як бінарний +)


Множення *


Ділення /
( '6' / '2' ); // 3, обидва операнди перетворюються на числа (не так, як бінарний +)


Остача від ділення %
 Результатом a % b є остача цілочислового ділення a на b
  alert( 5 % 2 ); // 1 - остача від ділення 5 на 2
  alert( 8 % 3 ); // 2 - остача від ділення 8 на 3
  alert( 8 % 4 ); // 0 - остача від ділення 8 на 4


Піднесення до степеня **.
 Піднесення до степеня a ** b множить a саме на себе b разів.
  alert( 2 ** 2 ); // 2² = 4
  alert( 2 ** 3 ); // 2³ = 8
  alert( 2 ** 4 ); // 2⁴ = 16
 Так само як у математиці, оператор піднесення також можна використовувати для дробових чисел.
 Наприклад, квадратний корінь це піднесення до степеня ½:
  alert( 4 ** (1/2) ); // 2 (степінь 1/2 — це теж саме, що квадратний корінь)
  alert( 8 ** (1/3) ); // 2 (степінь 1/3 — це теж саме, що кубічний корінь)


Пріоритет обчислення оператора:
Дужки перевизначають будь-який пріоритет, тому, якщо ми не задоволені типовим пріоритетом, ми можемо використовувати дужки, щоби змінити його. Наприклад: (1 + 2) * 2
