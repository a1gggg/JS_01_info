Підключення скриптів у HTML-документі:

1. З зовнішнього файлу: < sсript src="/path/to/script.js">< /sсript>
2. Зберігання в середині файлу html: < script> ... < /script>
3. З зовнішнього сайту:: < script src="https://cdnjs.cloudflare.com/ajax/libs/lodash.js/4.17.11/lodash.js">< /script>

Суворий режим “use strict”:

Для активації суворого режиму в JavaScript, ви можете використовувати директиву "use strict" або 'use strict'. Це дозволяє вам вказати, що ви хочете використовувати строгий режим для вашого коду. Суворий режим допомагає уникнути деяких помилок і небажаних поведінок у JavaScript.
"use strict";
    // цей код працюватиме у сучасному режимі
    ...

Змінні:

Створення:
    let x = 5; // змінна x має значення 5 (може бути змінена)
    var x = 5; // змінна x має значення 5 (може бути змінена)
    const x = 5; // константа x має значення 5 (не може бути змінена)

Виклик:
    alert(x); // виведе 5

Перезапис значення змінної:
let x; // оголошення змінної без ініціалізації

    x = 'Привіт!'; // присвоєння значення 'Привіт!
    alert(x); // виведе 'Привіт!'

    x = 'Світ!'; // значення змінено на 'Світ!' (змінна може бути зміненою, якщо вона створена за допомогою синтаксису let або var. Якщо змінна створена за допомогою const, то її значення не може бути змінено)
    alert(x); // виведе 'Світ!'
