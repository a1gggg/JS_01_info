Об'єкти

Об'єкт можна створити двома способами:

1. let user = new Object(); // синтаксис "конструктора об’єктів"
2. let user = {}; // синтаксис "літералу об’єкта"

При виконнані let user = new Object(); або let user = {}; створюється пустий об'єкт без ідентифікатора та значення.

Для створення об'єкту з ідентифікатором та значенням застосовують наступний синтаксис:

let user = {name: "John", age: 30}; // об'єкт з ідентифікатором name та age та значеннями "John" та 30.

(В даному випадку "John" відноситься до рядків (string), а 30 до числа (number)
